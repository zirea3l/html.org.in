HTML Favicon 📌

In this lecture, we are going to talk about some basic HTML favicon.

HTML Favicon

  • A favicon is a small image displayed next to the page title in the browser tab

Adding a Favicon in HTML

  • Use any image you like as your favicon. You can also create your own favicon on sites like https://www.favicon.cc.
  • A favicon image is displayed to the left of the page title in the browser tab.
  • Add a favicon to your website, either save your favicon image to the root directory of your webserver, or create a folder in the root directory.
